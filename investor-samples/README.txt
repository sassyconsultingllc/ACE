SASSY BROWSER - SAMPLE FILES DEMO PACK
======================================

This folder contains 30+ sample files demonstrating Sassy Browser's
universal file viewing capabilities. Just drag any file onto the
browser window or use File > Open.

NO ADOBE. NO MICROSOFT. NO SUBSCRIPTIONS. JUST SASSY.

Categories included:
- Documents (PDF, DOCX simulation, RTF)
- Spreadsheets (CSV, TSV)
- Code (Rust, Python, JavaScript, HTML, CSS)
- Data (JSON, XML, YAML, TOML)
- Scientific (PDB protein, MOL molecular, FASTA sequence)
- Images (PNG, JPG, SVG, WebP)
- Archives (would be ZIP, TAR)
- Config files (INI, ENV, CONF)
