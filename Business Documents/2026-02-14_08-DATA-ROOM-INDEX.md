- Private GitHub repository access
- Only for serious due diligence (term sheet stage)

## Data Room Platform Recommendations

| Platform | Cost | Best For |
|----------|:-----|---------|
| **DocSend** | $45/mo | Track who views what, analytics |
| **Google Drive** | Free | Simple, but limited tracking |
| **Notion** | Free–$10/mo | Organized, shareable, good UX |
| **Dealroom.co** | Custom | Purpose-built for fundraising |
| **Dropbox DocSend** | $45/mo | Industry standard for VCs |

**Recommendation:** Start with Google Drive (free), upgrade to DocSend when actively fundraising for analytics on investor engagement.

## Preparation Checklist

### Before First Investor Meeting
- [ ] All "✅ Ready" documents uploaded and organized
- [ ] Product demo video recorded (5 minutes)
- [ ] Screenshots taken on Windows, macOS, Linux
- [ ] Founder resume/CV updated
- [ ] Cap table created (even if single founder = 100%)
- [ ] NDA template ready for signing
- [ ] Bank statements gathered (last 6 months)

### Before Sending SAFE
- [ ] Attorney reviewed SAFE terms
- [ ] Valuation cap determined
- [ ] Pro rata rights decided
- [ ] Cap table updated with proposed dilution
- [ ] State blue sky law compliance confirmed
