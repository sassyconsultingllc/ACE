# Sassy Browser

**The only consumer browser not built on Google's code.**

---

## What It Is

A privacy-first web browser built from scratch in Rust. 12MB install. Zero telemetry. Opens 100+ file formats. Protects families. Fights trackers.

## Why It Matters

Every major browser (Chrome, Edge, Brave, Opera, Arc) runs Google's Chromium engine. When Google changes a policy, the entire browser market follows. Sassy is architecturally independent -- no Google code, no Google servers, no Google rules.

## What Makes It Different

| Feature | Chrome / Brave / Edge | Sassy |
|---------|----------------------|-------|
| Telemetry | Sends data to vendor | Zero -- no telemetry code exists |
| File viewer | ~10 formats | 100+ formats (PDF, DOCX, RAW, PDB, 3D models) |
| Install size | 300-400MB | 12MB |
| Ad blocker | Extension (killable by Google) | Built-in (immune to policy changes) |
| Family safety | Separate app/account | Built into the browser |
| Tracker defense | Block or nothing | Active poisoning + behavioral mimicry |
| Self-healing | No | Auto-repairs every 12 seconds |
| Password vault | Cloud-synced to vendor | Local-only, military-grade encryption |
