# Why Sassy Browser?

**A 3-minute read for people who care about the internet.**

## The Short Version

Chrome is free because you're the product. Sassy is free because software should just work without selling your data.

We built a browser from scratch -- not a Chrome clone, not a fork, not a skin. 76,996 lines of Rust. Zero telemetry. 12MB download. Opens any file format you throw at it.

---

## What You Get

### 1. Real Privacy

Not "incognito mode" privacy (which doesn't actually hide anything from Google). Real privacy. There is no telemetry code in the browser. Zero. We can't collect your data because the code to do it doesn't exist.

But we go further. When trackers try to fingerprint your browser (and they all do), we feed them fake data. They think they know who you are -- but they're building a profile of someone who doesn't exist.

### 2. Open Any File

PDF. DOCX. XLSX. RAW camera photos. 3D models. Chemical structures. Archives. Ebooks. Font files. 100+ formats, all built in. No plugins. No subscriptions. No "please install Adobe Acrobat."

That's $5,000+ per year in software you don't need to buy anymore.

### 3. Family-Safe

Adult, Teen, and Kid profiles built right in. No separate app. No Google Family Link account. No $9.99/month parental control subscription. Smart content detection keeps inappropriate material away from kids while letting adults browse normally.

### 4. It Fixes Itself

Every 12 seconds, the browser checks its own health. If tabs crash, it closes them. If memory spikes, it clears cache. If trackers surge, it reapplies protection. No other browser does this.

### 5. 25x Smaller

Chrome: 300MB download, 500MB+ RAM per tab, 15 background processes.
Sassy: 12MB download, <100MB RAM per tab, 1 process.

Works on old laptops. Works on slow internet. Works where Chrome doesn't.
